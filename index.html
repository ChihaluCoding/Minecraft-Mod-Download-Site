インストール手順

1. バージョンを確認してください

Minecraft クライアントがサポートされているバージョンと一致していることを確認します。

2. 依存関係を準備する

mod ファイルを mods フォルダーにコピーする前に、Fabric API などの必要なライブラリをダウンロードします。

3. ファイルをコピーして起動する

ダウンロードしたファイルを .minecraft/mods に配置し、ローダーを起動してインストールを確認します。
